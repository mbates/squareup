# Installation

## npm

```bash
npm install @bates-solutions/squareup square
```

## yarn

```bash
yarn add @bates-solutions/squareup square
```

## pnpm

```bash
pnpm add @bates-solutions/squareup square
```

## Peer Dependencies

The `square` SDK is a required peer dependency that provides the underlying API client.

### React Integration

If you're using React hooks and components:

```bash
npm install react
```

Supported React versions: `^18.0.0` or `^19.0.0`

### Angular Integration

If you're using Angular services:

```bash
npm install @angular/core @angular/common rxjs
```

Supported Angular versions: `^17.0.0`, `^18.0.0`, or `^19.0.0`

## TypeScript

This package is written in TypeScript and includes type definitions. TypeScript 5.0+ is recommended.

## Node.js

Requires Node.js 18 or higher.
